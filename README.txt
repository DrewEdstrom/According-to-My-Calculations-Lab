Description:

A calculator which can compute the result of expressions with addition,
subtraction, multiplication, and division operations. Grouping by
parentheses is also supported. Some things to be aware of:

- Single digits in expressions are supported, numbers of multiple digits
are not

- Expressions must contain a space between each character within the
expression
